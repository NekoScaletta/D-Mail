# Thư mục models

Sau khi chạy:

```bash
python training/train_model.py
```

Thư mục này sẽ có:

- `spam_model.pkl`
- `tfidf_vectorizer.pkl`

Hai file `.pkl` đang được ignore để tránh commit nhầm model lớn. Nếu cần deploy demo ngay trên Render, có thể bỏ ignore hoặc upload model theo quy trình riêng của nhóm.
